1st app	 : Bank
----------------------------------------------------
purpose : Utility helps you to create a bank account

interface: The Bank utility creates  specific files used by GasPump utility to  authenticate customers 

usage:   ./Bank -p   => start the bank application

			------------------------------
			DISPLAY A MENU
			------------------------------

			'A' : add a new customer in the bank
			'D' : remove  a customer 
			'R' : reinitialize the bank
			'L' : display all customers
			'I' : initialize database create a default customer 
			'E' : exit

	 ./Bank -r   => reinitialize the bank application
